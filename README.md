# Telegram бот-магазин

Минимальный Telegram-бот-магазин на `aiogram 3` с каталогом, корзиной и отправкой заказа администратору.

## Функции
- Главное меню с кнопками.
- Каталог товаров и добавление в корзину.
- Просмотр/очистка корзины.
- Оформление заказа (имя, телефон, адрес).
- Автоматическая отправка заказа администратору.

## Быстрый запуск
1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
2. Создайте `.env` на основе примера:
   ```bash
   cp .env.example .env
   ```
3. Укажите в `.env`:
   - `BOT_TOKEN` — токен бота из BotFather
   - `ADMIN_ID` — Telegram user id администратора
4. Запустите бота:
   ```bash
   python bot.py
   ```

## Как это работает
- Пользователь добавляет товары в корзину.
- Нажимает "Оформить заказ".
- Бот собирает контактные данные.
- Заказ отправляется в личные сообщения администратору.

## Кастомизация
Список товаров находится в `bot.py` в переменной `PRODUCTS`.
